Percent Selector

The percent selector bar is pretty straight forward. It allows for VERY easy creation of percent selection widgets like those below. They perform well (even, and especially on webkit-based mobile devices like iPhone and Android), are easily customized (they are "Image-free" -- all graphics are done dynamically with HTML5 canvas and CSS3), and are very easy to include in your pages.

Example 1 - Bare Bones

A "bare-bones" bar and can be created with the following markup:

<div class="percentSelector" style="width:500px; height:30px;"></div>

Example 2 - Initial Percent

Setting the initial percent is very easy!

<div class="percentSelector" style="width:500px; height:30px;" percent="50"></div>

Example 3 - Setting the Step

Setting the step allows you to define at what intervals values can be selected:

<div class="percentSelector" style="width:500px; height:30px;" step="10"></div>

Example 3 - Disabling User Interaction

If you don't want users to be able to modify a bar on the screen (e.g. if you only want to modify it programmatically), you can do so by simply setting 'enabled="false"':

<div class="percentSelector" style="width:500px; height:30px;" enabled="false" percent="73"></div>

Example 4 - Different Sizes

As mentioned above, you can easily create different sized versions of the bar using CSS:

<div class="percentSelector" style="width:40px; height:15px;" step="25"></div>
<div class="percentSelector" style="width:800px; height:75px;" step="2"></div>

Example 5 - Using the Callback

Adding javascript to the "onPercentChange" attribute (or adding a function to the object's "onpercentchange" property of the bar's DOM object) allows you to receive and process a callback when the percent changes. This callback only occurs when the user expects the percent to change (so, if a user is sliding the bar, they the event will not be issued until they let up on the mouse or their finger). Therefore, it's generally pretty safe to do things like make an AJAX call as the result of this event in order to persist the new value back to the server.

<script>
var percentUpdated = function(p) {
	$('#percentStatus').text("Percent: " + p + "%");
}
</script>
<div class="percentSelector" style="width:500px; height:30px; float:left;" onPercentChange="percentUpdated(percent)" step="5"></div>
<div id="percentStatus" style="float:left; font-size:28px; margin-left:5px;"></div>
